# ⚡ PRESCRIPTION SYSTEM - INSTALLATION

## 🎯 WHAT YOU NEED TO DO

The entire **Prescription Upload System** has been created and is ready to use.

All you need is **ONE command** to complete the setup:

---

## 📦 INSTALL COMMAND

```bash
cd c:\Users\DELL\pharmacy-ai-system\frontend
npm install tesseract.js
```

**Copy and paste this exactly into your Terminal.**

---

## ✅ WHAT THIS DOES

- Installs Tesseract.js (OCR library for text extraction)
- Downloads language models (~50MB)
- Enables prescription image processing
- All locally in browser (no server needed)

---

## 🚀 AFTER INSTALLATION

1. **Wait for npm to finish** (2-3 minutes)
2. **Refresh your browser**: `Ctrl+R` at `http://localhost:5173`
3. **Clear cache** (F12 → Application → Clear Storage)
4. **Go to Prescriptions** and test!
